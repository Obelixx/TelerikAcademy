Net beans, Eclipse,Adobe Dreamweaver , BlueJ, MonoDevelop

1. Net beans - netbeans.org
Net beans is written in Java and supports multiple Languages and Platforms with Large community.
It is free and open source.
It is made fo Java, but it supports HTML 5, XML, PHP, Groovy, C/C++, JavaScript also.
Netbeans can run on Windows, Linux and OS X(MAC).

2. Eclipse - eclipse.org
Standart package is IDE for Java development, but there are available Packagees for C/C++, PHP and others.
There is Maretplace for addons, solutions, IDE packages also. Documentation and open community is available too.

3. Adobe Dreamweaver - adobe.com/products/dreamweaver.html
It is paid product with trail available. Mainly you can build web pages with images and different styles.  It supports HTML5 and CSS.

4. BlueJ - bluej.org
Anather Java Development Environment used mainly for educational purposes(Java and Object-oriented programming). It can run under Windows, Linux, OS X and other. It runs With the help of JDK(Java Development Kit)

5. MonoDevelop - monodevelop.com
New for me, but maybe most interesting..
It is C# IDE supporting F#, Visual Basic .NET, C/C++ and Vala too. It runs under Linux, Windows and Mac OS X. Maybe it is best alternative of Visual Studio for Linux development.
